## Vue d'ensemble

Google AdMob est une plateforme de publicité mobile qui permet de générer des revenus à partir de votre application. En intégrant Firebase avec AdMob, vous accédez à des données d'utilisation et des fonctionnalités d'analyse supplémentaires.

## Formats d'annonces

| Format | Description | Recommandation |
| ------ | ----------- | -------------- |
| **Bannière** | Annonce rectangulaire standard en haut ou bas de l'écran | Débutant, non intrusif |
| **Interstitiel** | Annonce plein écran entre deux écrans | Transitions naturelles |
| **Native** | Annonce intégrée au design de l'app | Meilleure expérience utilisateur |
| **Avec récompense** | L'utilisateur gagne une récompense en regardant | Jeux, accès premium |

## Capacités clés

### Augmenter les revenus

- Diffusez des annonces de millions d'annonceurs Google
- Utilisez la médiation AdMob pour accéder à plus de 40 réseaux premium
- L'optimisation automatique ajuste les configurations pour maximiser les revenus

### Améliorer l'expérience utilisateur

- Les annonces natives s'intègrent au design de votre application
- Testez différentes mises en page et modèles d'annonces
- Contrôlez la fréquence et le placement

### Expansion mondiale

- Monétisez les utilisateurs dans plus de 200 marchés
- Auto-promotions pour la promotion croisée entre vos applications

### Rapports

AdMob génère des rapports spécifiques de monétisation pour éclairer votre stratégie produit.

## Niveaux d'intégration

### Niveau 1 : Métriques utilisateur AdMob

Accédez à des données et rapports dans la console AdMob.

### Niveau 2 : Liaison avec Firebase

Explorez vos données analytiques via la console Firebase. Liez votre compte AdMob à votre projet Firebase.

### Niveau 3 : SDK Firebase Analytics

Implémentez des analyses personnalisées et utilisez d'autres produits Firebase (Remote Config, A/B Testing) pour optimiser la monétisation.

## Parcours de démarrage

1. **Configurer** Firebase pour votre application
2. **Créer** un compte AdMob et enregistrer l'application
3. **Activer** les métriques utilisateur et lier AdMob à Firebase
4. **Ajouter** le SDK Google Mobile Ads
5. **Implémenter** la première annonce (commencer par une bannière)
6. **Optimiser** avec les formats avancés et les tests A/B
