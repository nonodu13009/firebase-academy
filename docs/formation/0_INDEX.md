## Le projet fil rouge

Tu vas construire **NoteFlow** : une app web de prise de notes collaborative.

A la fin, tu auras une vraie app en production avec :

- Inscription et connexion (Google, email)
- Creation, edition, suppression de notes (texte riche)
- Upload de fichiers joints (images, PDF)
- Partage de notes entre utilisateurs
- Notifications quand quelqu'un modifie une note partagee
- Un dashboard avec statistiques d'utilisation
- Le tout deploye en ligne, accessible par n'importe qui

**Stack** : Next.js 14+ (App Router) / TypeScript / Tailwind CSS / Firebase

## Prerequis

- Savoir coder en JavaScript/TypeScript (les bases)
- Avoir utilise React ou Next.js au moins une fois
- Un compte Google (gratuit)
- Node.js installe sur ta machine
- Un editeur de code (VS Code recommande)

Pas besoin de connaitre Firebase. C'est le but de cette formation.

## Comment suivre cette formation

**Regle 1** : Suis les niveaux dans l'ordre. Chaque niveau s'appuie sur le precedent.

**Regle 2** : Fais, ne lis pas. Chaque niveau contient des etapes concretes. Tape le code, teste, casse, repare.

**Regle 3** : Un niveau = une session. Ne fais pas tout d'un coup. Prends le temps de digerer chaque brique.

**Regle 4** : Si tu bloques, la doc de reference est dans le dossier `docs/` (les 24 fiches produit).

**Regle 5** : Un mot que tu ne comprends pas ? Consulte le [Glossaire](glossaire.md). Chaque terme technique est explique simplement, avec un lien pour approfondir.
